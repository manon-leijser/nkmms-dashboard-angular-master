Welcome to this Angular dashboard

In this application you can help Alliander prevent grid failures by judging partial discharge (PD) warnings and taking the appropriate follow-up actions.

The PD warnings are generated by our machine learning model.

The input for this model is PD data continuously measured by Alliander's more than 2300 Smart Cable Guard systems.

It has the following views:

Homepage The current view. Top ten warnings Here you can find the top ten warnings to be judged. Find warnings Here you can search for warnings by circuit-number, location or judgement-status. Annotate Here you can judge the warnings. There is a form you can fill in with your judgement.
